Featured Project

Agentic QA Tool

A human-in-the-loop QA assistant that plans test cases, asks for reviewer approval, generates runnable tests, executes them, and publishes report artifacts.

Open GitHub Repository

What It Does

Interactive Planning

Lets the user choose strategy scope like smoke, sanity, regression, or custom notes before planning starts.

Approval Before Execution

Displays proposed test cases and requires explicit approve/edit/add actions before generating or running tests.

Evidence Output

Produces generated test scripts and markdown/html reports to keep execution outcomes reviewable and reproducible.

Learning Progress

This project has been a hands-on learning path for agent orchestration in QA automation. I started with a linear LLM workflow, then evolved it into a provider-agnostic, human-in-the-loop system with approval gates and cleanup tooling for generated artifacts.

Current focus: improving test quality controls, reducing noisy outputs, and making the generated evidence easier for hiring managers and teams to evaluate quickly.
